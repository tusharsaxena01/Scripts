## Python

- Create a virtual environment (Recommended)

    `
    python -m venv venv
    `
- Activate Python environment
  - Windows
  `venv/Script/activate`
  - Linux
  `source venv/bin/activate`
- Installed Required modules

  `pip install -r ./requirements.txt`

- Deactivate the Environment

  - Windows
  `deactivate`
  - Linux
  `deactivate`
